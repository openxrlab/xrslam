Data Preparation¶

EuRoC Dataset contains stereo images, synchronized IMU measurements, and accurate motion and structure ground truth. In XRSLAM, we only use Monocular camera and IMU data. The bottom figures are the representative images of the Machine hall and Vicon room, where the EuRoC dataset was collected.

It is recommended to construct your folder structure like this:

data
├── EuRoC
│   ├── MH_01_easy
│           |── mav0
│                |──...
│   └── MH_02_easy
│   └── ...

Take EuRoC as an example, download a sequence (ASL format) and uncompress it. You can also download the compressed zip file through the link below.
